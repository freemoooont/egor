# Описание макетов Figma

Источник: [Figma — Untitled](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=0-1)

Приложение **Micocards** — сервис для обучения с помощью флеш-карточек: ИИ-генерация колод, ручное создание, режим практики с отслеживанием прогресса.

Ниже — все уникальные экраны из файла. Большинство представлены в десктопной (1440px) и мобильной (390px) версиях.

---

## 1. Авторизация — Вход

Страница входа с логотипом, переключателем табов «Вход / Регистрация», полями email и пароля и кнопкой «Продолжить».

- [Десктоп — пустая форма (1:674)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-674) — базовое состояние, кнопка неактивна (бледно-оранжевая).
- [Десктоп — заполненная форма (1:740)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-740) — пароль введён, кнопка «Продолжить» активна.
- [Десктоп — ошибка валидации email (1:760)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-760) — поле email с красной рамкой, состояние ошибки.
- [Мобильный — пустая форма (1:695)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-695) — мобильная версия экрана входа.

## 2. Авторизация — Регистрация

- [Десктоп — таб «Регистрация» (1:719)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-719) — переключатель в положении «Регистрация», поля email и пароля, неактивная кнопка «Продолжить».

## 3. Главная — «Мои колоды»

Список созданных пользователем колод. Карточки содержат название, количество терминов/уроков, автора и контекстное меню.

- [Десктоп — сетка колод (1:816)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-816) — три колонки карточек, счётчик колод рядом с заголовком.
- [Мобильный — список колод (1:852)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-852) — одноколоночный список со скроллом.

## 4. Настройки аккаунта

Управление профилем: загрузка аватара, имя, email с возможностью редактирования, кнопки «Выйти из аккаунта» и «Сменить пароль».

- [Десктоп — настройки (1:891)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-891) — широкая компоновка, кнопки в строку.
- [Мобильный — настройки (1:939)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-939) — карточки в один столбец, кнопки на всю ширину.

## 5. Создание колоды

Форма создания колоды: поле названия, кнопка ИИ-генерации, нумерованный список карточек «Термин — Определение» с drag-and-drop и удалением, кнопки «Добавить карточку», «Создать» и «Создать и практиковать».

- [Десктоп — создание (1:990)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-990) — кнопки сохранения справа сверху, термин и определение в одну строку.
- [Мобильный — создание (1:1054)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-1054) — кнопки сохранения раздельно сверху, поля термина и определения вертикально.

## 6. Просмотр и практика колоды

Большая флеш-карточка с термином, навигация между карточками, тумблер «Отслеживать прогресс», список всех терминов с определениями ниже.

- [Десктоп — без отслеживания (1:1122)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-1122) — тумблер выключен, только стрелки навигации «1/32».
- [Десктоп — с отслеживанием (1:1473)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-1473) — тумблер включён, появились кнопки оценки «Не знаю / Ещё изучаю / Знаю».
- [Мобильный — без отслеживания (1:1202)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-1202) — мобильная версия с выключенным тумблером.
- [Мобильный — с отслеживанием (1:1285)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-1285) — мобильная версия с активным тумблером и кнопками самооценки.

## 7. Результаты практики

Итоги прохождения колоды: заголовок «Так держать! Продолжайте в том же духе!», карточка «Ваши успехи» с круговой диаграммой прогресса и легендой «Знаю / Ещё изучаю / Не знаю», кнопки «Пройти карточки заново» и «На главную».

- [Десктоп — результаты (1:1369)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-1369) — диаграмма, легенда и кнопки в одну строку.
- [Мобильный — результаты (1:1419)](https://www.figma.com/design/FDkPi2WrDztxvhQeWQtgWo/Untitled?node-id=1-1419) — диаграмма сверху, легенда и кнопки столбцом.
